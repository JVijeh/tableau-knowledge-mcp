## üéØ Tips for Effective Queries

1. **Be Specific**: Reference specific books if you know them
   - "Search my 'Learning Tableau 2025' book for..."

2. **Combine Theory + Practice**: Ask for both explanation and application
   - "Explain X from my books, then demonstrate with my data"

3. **Request Examples**: Ask for concrete examples
   - "Show me 3 different ways to calculate..."

4. **Iterate**: Build on previous answers
   - "Now apply that same technique to a different field"

5. **Ask for Comparisons**: Compare multiple approaches
   - "Show me the difference between method A and method B"

## üêõ If Queries Don't Work

**Knowledge Base Returns Nothing:**
- Check if PDFs were indexed correctly
- Verify server is running (üîå icon)
- Try simpler query: "List all books"

**Tableau Returns Errors:**
- Verify PAT credentials in config
- Check server URL and site name
- Test with simpler query: "List datasources"

**Both Servers Show as Disconnected:**
- Restart Claude Desktop completely
- Check configuration file paths
- Review logs for error messages

See [TROUBLESHOOTING.md](../docs/TROUBLESHOOTING.md) for detailed help.
